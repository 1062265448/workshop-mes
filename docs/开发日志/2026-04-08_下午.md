## 📝 技术要点

### 1. Prisma 一对多关系
```prisma
model Meeting {
  tasks MeetingTask[] @relation("MeetingTasks")
}

model MeetingTask {
  meetingId Int
  meeting   Meeting @relation(fields: [meetingId], references: [id], onDelete: Cascade)
}
```

### 2. NestJS 级联删除
```typescript
@Delete(':id')
async deleteMeeting(@Param('id') id: number) {
  // Prisma 会自动删除关联的 participants/tasks/attachments
  return this.prisma.meeting.delete({ where: { id } });
}
```

### 3. Vue 表单验证
```typescript
const meetingRules = {
  title: [{ required: true, message: '请输入会议主题', trigger: 'blur' }],
  startTime: [{ required: true, message: '请选择开始时间', trigger: 'change' }],
  endTime: [{ required: true, message: '请选择结束时间', trigger: 'change' }],
}
```

### 4. 统计查询优化
```typescript
const [total, completed, pendingTasks, overdueTasks] = await Promise.all([
  this.prisma.meeting.count({ where }),
  this.prisma.meeting.count({ where: { ...where, status: 'completed' } }),
  this.prisma.meetingTask.count({ where: { meeting: where, status: { not: 'done' } } }),
  this.prisma.meetingTask.count({ where: { meeting: where, status: { not: 'done' }, dueDate: { lt: new Date() } } }),
]);
```
